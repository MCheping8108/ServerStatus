# ServerStatus

English / [简体中文](./README-CN.md)

This plugin can display Minecraft Server's status and info on website

## How can I use this?

Too Easy! Install this plugin and run server, use your browser open your ServerStatus website, `http://yourip:8080`. Also, you can change port number

## commands

if your server has running, you can change the port in the ***config.yml*** and in console typing `/ssreload`
